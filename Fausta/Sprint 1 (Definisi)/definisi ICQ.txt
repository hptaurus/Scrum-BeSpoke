ICQ (“I Seek You”)
ICQ adalah sebuah system pesan instan populer yang memungkinkan para pengguna internet untuk saling bertukar pesan. http://www.icq.com
,merupakan aplikasi popular untuk mengirim pesa secara online sepeti AIM dan Yahoo Messenger, tetapi memungkinkan usr untuk membuat Chat room,dan meng-chat dengan beberapa user dalam waktu yang sama.
ICQ merupakan program chat bersifat "community-Oriented" dari pada program chat lain.
Sejak aplikasi dirilis pada 1996, ICQ telah direvisi berkali-kali. dimulai dari chat dasar menjadi tool komunikasi yang mengizinkan user untuk komunikasi dengan berbagai cara. seperti ICQ sekarang menyediakan 
komunikasi suara, video dan VOIP, dan juga bisa digunakan untuk mengirimkan teks ke Handphone melalui SMS, aplikasi seperti ICQ2Go,The ICQ Toolbar, dan ICQmail, telah dijadikan sebagai tool tambahan dalam ICQ
ICQ dinamakan community-oriented dan memberikan beberapa jenis chat room, dengan kategori: umur, lokasi, life style, belief dan klasifikasi lain. user bisa menjoin chat room dengan menggunakan software ICQ atau melalui web pada ICQ.com
